# basic_info_site
practice site for node js server

## how to run

1. make sure you have node and npm installed
2. run ``npm install`` to install dependencies
3. use ``PORT`` enviroment variable to specify which port server will listen on. Default is 3000
4. run ``npx tsc`` to compile typescript into javascript into ``./server/`` folder
5. run ``node ./server/index.js`` to start the server and access the site on ``http://localhost:3000/``
